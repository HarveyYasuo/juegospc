Política de Privacidad para Analizador de Sitios para AdSense

Fecha de vigencia: 05/08/2025

1. Introducción

Bienvenido a Analizador de Sitios para AdSense (la "Verificar Web Adsense"). Esta aplicación es proporcionada por Harvey Rivas. Nos tomamos tu privacidad en serio. Esta política describe qué información procesamos y cómo la utilizamos.

2. Información que Procesamos

La Aplicación está diseñada para funcionar con una mínima recopilación de datos. No recopilamos información de identificación personal como tu nombre, dirección de correo electrónico o información de contacto.

La única información que procesamos activamente es la siguiente:

  • URL del Sitio Web: La dirección del sitio web que ingresas voluntariamente en la aplicación para su análisis.

3. Cómo Utilizamos la Información

La URL que proporcionas se utiliza con el único propósito de realizar el análisis solicitado. Este proceso implica:

  1. Realizar comprobaciones técnicas automatizadas sobre el contenido público de la URL proporcionada.
  2. Enviar el contenido de texto público de la URL a un servicio de inteligencia artificial de terceros (API de Google Gemini) para un análisis cualitativo.

No almacenamos, guardamos ni compartimos la URL que ingresas para ningún otro propósito. El procesamiento es transaccional y se realiza en tiempo real.

4. Servicios de Terceros

Para proporcionar el análisis de contenido, la Aplicación se comunica con los siguientes servicios de terceros:

  • Tu Servidor Backend: La URL que ingresas se envía a nuestro servidor seguro, que actúa como intermediario para proteger la clave de la API.
  • API de Google Gemini: El contenido de texto de la URL se envía a la API de Google para su análisis. El uso que Google hace de estos datos se rige por la Política de Privacidad de Google.

5. Seguridad

Hemos implementado medidas de seguridad para proteger la información durante el tránsito. La comunicación entre la aplicación y nuestro servidor, así como entre nuestro servidor y la API de Google, se realiza a través de conexiones seguras (HTTPS).

6. Cambios a esta Política de Privacidad

Podemos actualizar nuestra Política de Privacidad de vez en cuando. Te notificaremos cualquier cambio publicando la nueva Política de Privacidad en esta página. Se te aconseja revisar esta Política de Privacidad periódicamente para cualquier cambio.
